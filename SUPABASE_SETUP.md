# Supabase User Monitoring Setup Guide
# рЃАрЃљрЃ«рЃћрЃџрЃЏрЃФрЃдрЃЋрЃљрЃюрЃћрЃџрЃЮ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃЌрЃљ рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃЊрЃљрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ

## ­ЪЊІ рЃАрЃљрЃГрЃўрЃарЃЮ рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃў
- Supabase рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў ([supabase.com](https://supabase.com))
- Python рЃњрЃљрЃарЃћрЃЏрЃЮ (рЃБрЃЎрЃЋрЃћ рЃњрЃљрЃЦрЃЋрЃЌ)
- .env рЃцрЃљрЃўрЃџрЃў

## ­Ъџђ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃАрЃљрЃцрЃћрЃ«рЃБрЃарЃћрЃЉрЃў

### 1. Supabase рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ

1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ [supabase.com](https://supabase.com) рЃЊрЃљ рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃљрЃюрЃњрЃљрЃарЃўрЃерЃў
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "New Project"
3. рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ:
   - **Name**: `portfolio-analytics` (рЃљрЃю рЃАрЃљрЃАрЃБрЃарЃЋрЃћрЃџрЃў рЃАрЃљрЃ«рЃћрЃџрЃў)
   - **Database Password**: рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃФрЃџрЃўрЃћрЃарЃў рЃърЃљрЃарЃЮрЃџрЃў
   - **Region**: рЃљрЃўрЃарЃЕрЃўрЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃљрЃю рЃљрЃ«рЃџрЃЮрЃА рЃЏрЃЊрЃћрЃЉрЃљрЃарЃћ рЃарЃћрЃњрЃўрЃЮрЃюрЃў
4. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "Create new project"

### 2. рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃЦрЃћрЃЏрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

1. рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃерЃў рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ **SQL Editor**
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "New query"
3. рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ рЃЊрЃљ рЃЕрЃљрЃАрЃЋрЃўрЃЌ `supabase_schema.sql` рЃцрЃљрЃўрЃџрЃўрЃА рЃЏрЃЌрЃћрЃџрЃў рЃерЃўрЃюрЃљрЃљрЃарЃАрЃў
4. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ "Run" - рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА "Success"

### 3. API рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЏрЃўрЃдрЃћрЃЉрЃљ

1. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ **Settings** Рєњ **API**
2. рЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћрЃЌ:
   - **Project URL**: `https://xxxxx.supabase.co`
   - **anon/public key**: `eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...`

### 4. .env рЃцрЃљрЃўрЃџрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃФрЃўрЃарЃерЃў (рЃЌрЃБ рЃљрЃа рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА):

```bash
# рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЌ рЃћрЃА рЃ«рЃљрЃќрЃћрЃЉрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃљрЃарЃАрЃћрЃЉрЃБрЃџ .env рЃцрЃљрЃўрЃџрЃерЃў
```

### 5. рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ

### 6. рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ

1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃАрЃљрЃўрЃбрЃў рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў
2. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃљрЃЊрЃЏрЃўрЃюрЃўрЃА рЃърЃљрЃюрЃћрЃџрЃерЃў: `/admin/analytics`
3. рЃБрЃюрЃЊрЃљ рЃюрЃљрЃ«рЃЮрЃЌ рЃљрЃюрЃљрЃџрЃўрЃбрЃўрЃЎрЃўрЃА рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃў

## ­ЪћЇ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў

### рЃЋрЃўрЃќрЃўрЃбрЃћрЃЉрЃўрЃА рЃбрЃарЃћрЃЎрЃўрЃюрЃњрЃўрЃА рЃбрЃћрЃАрЃбрЃў:
1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃАрЃљрЃўрЃбрЃў рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃћрЃЉрЃерЃў
2. рЃЊрЃљрЃљрЃГрЃўрЃарЃћрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЉрЃљрЃарЃљрЃЌрЃћрЃЉрЃА
3. рЃњрЃљрЃЊрЃљрЃЊрЃўрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃќрЃћ
4. рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ `/admin/analytics` - рЃБрЃюрЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃА рЃЋрЃўрЃќрЃўрЃбрЃћрЃЉрЃў

### API рЃбрЃћрЃАрЃбрЃў:
```bash
# рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ curl-рЃўрЃЌ
curl -X POST http://localhost:5002/api/track-visit \
  -H "Content-Type: application/json" \
  -d '{"page_url":"test","screen_resolution":"1920x1080"}'
```

## ­ЪЊі рЃарЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃўрЃЎрЃарЃўрЃЉрЃћрЃЉрЃљ

- **IP рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃў**: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃарЃћрЃљрЃџрЃБрЃарЃў IP
- **рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃарЃћрЃќрЃЮрЃџрЃБрЃфрЃўрЃљ**: `1920x1080`, `1366x768` рЃЊрЃљ рЃљ.рЃе.
- **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃљрЃњрЃћрЃюрЃбрЃў**: рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ
- **рЃњрЃЋрЃћрЃарЃЊрЃўрЃА URL**: рЃарЃЮрЃЏрЃћрЃџ рЃњрЃЋрЃћрЃарЃЊрЃА рЃАрЃбрЃБрЃЏрЃарЃЮрЃЉрЃА
- **рЃарЃћрЃцрЃћрЃарЃћрЃарЃў**: рЃАрЃљрЃўрЃЊрЃљрЃю рЃЏрЃЮрЃЋрЃўрЃЊрЃљ
- **рЃАрЃћрЃАрЃўрЃўрЃА ID**: рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃАрЃћрЃАрЃўрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃбрЃЮрЃарЃў
- **рЃЊрЃарЃЮрЃўрЃА рЃерЃбрЃљрЃЏрЃърЃў**: рЃЋрЃўрЃќрЃўрЃбрЃўрЃА рЃЊрЃарЃЮ

## ­Ъћњ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ

- **рЃљрЃюрЃЮрЃюрЃўрЃЏрЃБрЃарЃў рЃбрЃарЃћрЃЎрЃўрЃюрЃњрЃў**: рЃљрЃа рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃърЃўрЃарЃљрЃЊрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў
- **Row Level Security**: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃЊрЃљрЃфрЃБрЃџрЃўрЃљ Supabase-рЃерЃў
- **GDPR рЃЌрЃљрЃЋрЃАрЃћрЃЉрЃљрЃЊрЃў**: рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЏрЃЮрЃарЃЌрЃБрЃџрЃў рЃўрЃДрЃЮрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃЏрЃЮрЃ«рЃАрЃюрЃўрЃЌ

## ­ЪЏа рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃњрЃЋрЃљрЃарЃћрЃЉрЃљ

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "Supabase credentials not configured"
**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ**: рЃерЃћрЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ SUPABASE_* рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "Table doesn't exist"
**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ**: рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ `supabase_schema.sql` Supabase SQL Editor-рЃерЃў

### рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃљ: "No data in analytics"
**рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ**: рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃАрЃљрЃўрЃбрЃў рЃЊрЃљ рЃЊрЃљрЃћрЃџрЃЮрЃЊрЃћрЃЌ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЋрЃўрЃќрЃўрЃбрЃА

## ­ЪЊѕ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃюрЃљрЃЉрЃўрЃ»рЃћрЃЉрЃў

рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЏрЃќрЃљрЃЊрЃљрЃљ! рЃљрЃ«рЃџрЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ:

1. **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў** `/admin/analytics`-рЃерЃў
2. **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃћрЃЦрЃАрЃърЃЮрЃарЃбрЃў** Supabase рЃЊрЃћрЃерЃЉрЃЮрЃарЃЊрЃўрЃЊрЃљрЃю
3. **рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЏрЃћрЃбрЃарЃўрЃЎрЃћрЃЉрЃў** рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃЌ
4. **рЃњрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃћрЃЉрЃў** рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃЏрЃЮрЃЋрЃџрЃћрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
