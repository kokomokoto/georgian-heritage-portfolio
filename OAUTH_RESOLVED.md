## ­ЪћД OAuth Google/Facebook-рЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ (рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў):

### рЃЏрЃљрЃарЃбрЃўрЃЋрЃў 15-рЃгрЃБрЃЌрЃўрЃљрЃюрЃў Setup:

1. **Google Cloud Console**: https://console.cloud.google.com/
   - рЃърЃарЃЮрЃћрЃЦрЃбрЃў > APIs & Services > Credentials
   - OAuth 2.0 Client ID > Web Application
   - Redirect URI: `http://localhost:5001/auth/google/callback`

2. **Facebook Developers**: https://developers.facebook.com/
   - Create App > Consumer 
   - Facebook Login > Settings
   - Valid OAuth Redirect URIs: `http://localhost:5001/auth/facebook/callback`

3. **.env рЃцрЃљрЃўрЃџрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ**:
   ```env
   GOOGLE_CLIENT_ID=рЃЌрЃЦрЃЋрЃћрЃюрЃў-google-client-id
   GOOGLE_CLIENT_SECRET=рЃЌрЃЦрЃЋрЃћрЃюрЃў-google-client-secret
   FACEBOOK_CLIENT_ID=рЃЌрЃЦрЃЋрЃћрЃюрЃў-facebook-app-id
   FACEBOOK_CLIENT_SECRET=рЃЌрЃЦрЃЋрЃћрЃюрЃў-facebook-app-secret
   ```

4. **рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃарЃћрЃАрЃбрЃљрЃарЃбрЃў**:
   ```powershell
   # Ctrl+C (рЃњрЃљрЃЕрЃћрЃарЃћрЃЉрЃљ)
   & "C:/Users/ATA/Desktop/saitis gashveba/4/.venv/Scripts/python.exe" app.py
   ```

## ­Ъџђ рЃЏрЃБрЃерЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў (OAuth-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ):

## ­ЪјЅ рЃерЃћрЃЊрЃћрЃњрЃў:

**рЃЏрЃБрЃерЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ OAuth-рЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ! рЃАрЃЮрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃљрЃарЃљрЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ рЃцрЃБрЃюрЃЦрЃфрЃўрЃљрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф 15 рЃгрЃБрЃЌрЃерЃў рЃЕрЃљрЃўрЃарЃЌрЃЋрЃћрЃЉрЃљ рЃАрЃБрЃарЃЋрЃўрЃџрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў.**

- РюЁ рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃЌ
